The new `/deep-planning` command transforms Agentforce into a thoughtful architect that deeply plans before it builds. Focus Chain brings structure and visibility to your tasks. It automatically generates and tracks to-do lists as Agentforce works, displaying real-time progress right in the header. Each step updates dynamically as tasks are completed, and you can edit the list at any time.

It also includes a Smart Reminder System to help keep Agentforce's work on track:
- A configurable task reminder interval
- Automatic reminders when switching from Plan Mode to Act Mode
- Optional manual reminders when to-do lists are edited

Together, Deep Planning and Focus Chain make long-horizon development easier to manage and keep your work organized from initial design to final commit.
